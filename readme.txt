== Description ==

This plugin displays the Kent FA news feed as a widget.

The news is pulled from the Kent FA RSS feed every 15 minutes and cached locally. The look of the plugin is determined by the CSS file and background image in the plugin directory.

== Installation ==

1. Upload the zip file contents to the `/wp-content/plugins/kentfa` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your widgets section, drag the Kent FA widget to the desired place

== Frequently Asked Questions ==

= How often does the feed get recached? =

Every 15 minutes.

= How can I change the appearance of the widget? =

The best way is to edit the image and CSS in the plugin directory.
